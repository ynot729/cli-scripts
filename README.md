## Helpful CLI Scripts

A collection of scripts to make life easier

## Install

Clone the repo and add the bin to your path

```bash
# in my case i add this to ~/.zshrc
export PATH="$HOME/Documents/Dev/cli-scripts/bin:$PATH"
```
